# pomodoro-timer
Pomodoro Study Timer Website

🍅 Pomodoro Timer Web App

A simple and clean Pomodoro Timer built to help you stay focused and productive using the Pomodoro Technique.

📌 Features
⏱️ 25-minute focus timer (Pomodoro session)
☕ Short break timer (5 minutes)
🔄 Start, pause, and reset functionality
